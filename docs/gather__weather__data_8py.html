Detailed Description

Periodically fetches and logs telemetry data from ThingsBoard.

This script authenticates with the ThingsBoard Cloud API using JWT, retrieves telemetry values (temperature and humidity), and appends them to a daily CSV log. It supports automatic token refresh and runs as a continuous loop at defined intervals.
