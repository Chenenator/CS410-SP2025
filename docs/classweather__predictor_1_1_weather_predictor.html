Detailed Description

Predicts future weather (max/min temperatures) using Ridge regression.

The class loads and preprocesses weather data, creates lagging and rolling features, fits a Ridge regression model, and performs sequential prediction for a specified number of days.

  • Loads weather data into a DataFrame.
